## Git Flow 규칙

| 태그       | 설명                                         |
|-----------|--------------------------------------------|
| `feat`    | 새로운 기능 추가                            |
| `fix`     | 버그 수정                                   |
| `build`   | 빌드 관련 파일 수정 / 모듈 설치 또는 삭제  |
| `chore`   | 코드 변경이 없는 기타 자잘한 수정          |
| `ci`      | CI 관련 설정 수정                          |
| `docs`    | 문서 수정 (README, 주석 추가 등)           |
| `style`   | 코드 스타일 및 포맷 변경 (공백, 세미콜론 등) |
| `refactor`| 기능 변화 없이 코드 구조 개선              |
| `test`    | 테스트 코드 추가 또는 수정                 |
| `perf`    | 성능 개선                                  |
